# Data Folder

Place your dataset files here before running the API.

## Required Files

| File | Dataset | Download |
|------|---------|----------|
| `Reviews1.csv` | Amazon Fine Food Reviews | [Download from Kaggle](https://www.kaggle.com/datasets/snap/amazon-fine-food-reviews) |
| `Reviews2.json` | Amazon Electronics Reviews | [Download from Kaggle](https://www.kaggle.com/datasets/snap/amazon-reviews) |

## After Downloading

1. Rename the food reviews file to `Reviews1.csv`
2. Rename the electronics reviews file to `Reviews2.json`
3. Place both files in this `data/` folder
4. Run `python main.py` from the project root

## Note
These files are not included in the repository due to their large size.
This folder is intentionally kept empty — add your own dataset files here.
